Usage: mainbatti-talika [FILE] [OPTION...]
Read xml FILE for loadshedding data

Commandline Options:
    -g<N>           select group N (1-7)
    -12, --twelve   enable 12 hour mode
    -n,  --nepali   change language to nepali
    -h,  --help     display help message

Application Options:
    <N>     Highlight group N (1-7)
    t       toggle 12 & 24 hour modes
    n       change language to nepali
    e       change language to english
    c       toggle background color
    q       quit the application
